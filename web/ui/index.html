Architecture

Choreographed agents on a RabbitMQ topic exchange.

  1. ① gateway → incident.created
  2. ② retriever → DNS + hybrid RAG → context.assembled
  3. ③ reasoner → LLM (BYO) → analysis.completed
  4. ④ validator → schema + grounding + injection → analysis.validated
  5. ⑤ executor → idempotent actions → action.executed
